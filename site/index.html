Designed for terminal playback

          
Fast where the frame loop actually hurts.

        

        

          

            01
            
Vectorized pixels

            

              Frames are resized with OpenCV and mapped through NumPy arrays, avoiding
              per-pixel Python loops.
            

          

          

            02
            
ANSI-native rendering

            

              The renderer returns the cursor home with escape sequences instead of clearing
              the full screen every frame.
            

          

          

            03
            
Frame pacing

            

              A monotonic clock respects source FPS and skips sleeping when processing falls
              behind.
            

          

        

      


      

        

          

            
Live browser simulation

            
The product signal is the terminal.

            

              This page does not fake a video player. It previews the shape of the real CLI:
              dense ASCII, a stable output surface, and motion that reads cleanly at terminal
              scale.
            

          

          

            

              $
              ascii-motion input.mov --fit-terminal
            

            

              >
              source_fps=29.97 width=132 charset=classic
            

            

              >
              render=ANSI alternate_screen=true cursor=home
            

          

        

      


      

        

          
Pipeline

          
A small loop with strict responsibilities.

        

        

          VideoCapture
          Resize
          Luminance
          LUT ASCII
          ANSI Render
        

        

          

            Rec. 709
            0.2126R + 0.7152G + 0.0722B
          

          

            0 full clears
            after startup, every frame uses cursor repositioning
          

          

            30/60 FPS
            source-aware pacing with optional override

              Clone the project, install it in editable mode, and point the CLI at a video
              file or a camera index.

source .venv/bin/activate
