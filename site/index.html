Product surface

          
Built like a terminal media tool, not a one-off script.

          

            The CLI now covers playback, controls, visual processing, export, and package-grade
            validation while keeping the runtime dependency set focused.
          

        


        

          

            Playback
            
HUD, progress, pause and seek

            

              Compact playback state is visible by default, with configurable controls for quit,
              pause/resume, seeking, and help.
            

          

          

            Export
            
Frames, text and ANSI files

            

              Save one frame, a form-feed separated text animation, ANSI playback output, or a
              directory of numbered frame files.
            

          

          

            Visuals
            
ASCII, edges, hybrid and dithering

            

              Luminance mapping is joined by Sobel edge emphasis, hybrid rendering, and ordered
              Bayer dithering.
            

          

          

            Color
            
Plain, truecolor, 256 and grayscale

            

              Choose plain text for speed or ANSI color output for richer terminal captures and
              exports.
            

          

          

            Timing
            
FPS pacing and real-time mode

            

              Respect source FPS by default, override playback FPS when needed, or skip frames to
              preserve wall-clock timing.
            

          

          

            Packaging
            
Homebrew-ready CLI

            

              Install from a tap, run as `ascii-motion`, or use `python -m ascii_motion` during
              development.
            

          

        

      


      

        

          

            
Interactive playback

            
Playback controls belong in the terminal.

            

              The app stays stdout-native, but it behaves like a focused media player: pause,
              resume, seek, inspect progress, and keep rendering smooth without a GUI layer.
            

          

          

            
qquit

            
spacepause / resume

            
←seek backward

            
→seek forward

            
hbackward fallback

            
lforward fallback

            
?toggle help

          

        

      


      

        

          
Non-interactive workflows

          
Export for docs, demos, snapshots and terminal playback.

        

        

          

            
Preview metadata

            
ascii-motion video.mp4 --preview

            
Inspect source FPS, frame count, duration, selected dimensions and visual settings.

          

          

            
Save one frame

            
ascii-motion video.mp4 --frame-at 12.5 --width 120 > frame.txt

            
Render exactly one timestamp to stdout and redirect it wherever you need.

          

          

            
Export an animation

            
ascii-motion video.mp4 --export output.txt --duration 5

            
Write a plain text animation using form-feed separators between frames.

          

          

            
Export ANSI color

            
ascii-motion video.mp4 --export-ansi output.ans --color 256

            
Create terminal-replayable output with ANSI cursor-home and color sequences.

          

        

      


      

        

          
Pipeline

          
A tight loop with explicit performance boundaries.

        

        

          VideoCapture
          Resize
          Luminance / Edges
          Dither + LUT
          ANSI Render / Export
        

        

          

            Rec. 709
            0.2126R + 0.7152G + 0.0722B over BGR OpenCV frames
          

          

            No pixel loops
            NumPy handles full matrices; Python only assembles text by row
          

          

            No frame clears
            rendering uses ANSI cursor-home instead of clearing the screen per frame
          

          

            Real-time mode
            optional frame skipping keeps playback aligned with wall-clock time
